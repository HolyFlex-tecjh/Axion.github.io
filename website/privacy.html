1. Introduktion

Denne privatlivspolitik beskriver, hvordan Axion Bot ("vi", "os", "vores") indsamler, bruger og beskytter dine personlige oplysninger, når du bruger vores Discord bot og relaterede tjenester.

Ved at bruge Axion Bot accepterer du vilkårene i denne privatlivspolitik. Hvis du ikke er enig i disse vilkår, bedes du ikke bruge vores tjenester.

2. Hvilke Data Indsamler Vi

Vi indsamler følgende typer af information:

2.1 Discord Data

  • Bruger ID og brugernavn
  • Server ID og servernavn
  • Kanal ID'er hvor botten er aktiv
  • Beskedindhold kun når det er nødvendigt for bot funktionalitet
  • Rolle information

2.2 Brugsdata

  • Kommandoer brugt og tidspunkter
  • Bot interaktioner og præferencer
  • Fejllogger og performance data

2.3 Webstedsdata

  • IP-adresse
  • Browser information
  • Besøgsstatistikker
  • Cookies og lignende teknologier

3. Hvordan Bruger Vi Dine Data

Vi bruger dine data til følgende formål:

3.1 Tjenesteudbydelse

  • Levering af bot funktionalitet
  • Behandling af kommandoer
  • Opbevaring af server indstillinger
  • Moderation og sikkerhed

3.2 Forbedring af Tjenester

  • Analyse af brugsstatistikker
  • Fejlfinding og optimering
  • Udvikling af nye funktioner

3.3 Kommunikation

  • Support og kundeservice
  • Vigtige opdateringer om tjenesten
  • Markedsføring (kun med samtykke)

4. Deling af Data

Vi deler ikke dine personlige data med tredjeparter, undtagen i følgende tilfælde:

  • Juridiske krav: Når det kræves af lov eller retskendelse
  • Sikkerhed: For at beskytte vores brugere og tjenester
  • Tjenesteudbydere: Med betroede partnere der hjælper med at levere vores tjenester
  • Samtykke: Når du har givet eksplicit samtykke

Vigtig note: Vi sælger aldrig dine personlige data til tredjeparter til kommercielle formål.

5. Opbevaring og Sletning af Data

5.1 Opbevaringsperioder

  • Server indstillinger: Indtil botten fjernes fra serveren
  • Brugerdata: Indtil brugeren anmoder om sletning
  • Logfiler: Maksimalt 90 dage
  • Backup data: Maksimalt 30 dage

5.2 Automatisk Sletning

Data slettes automatisk når:

  • Botten fjernes fra en server
  • En bruger forlader alle servere med botten
  • Opbevaringsperioden udløber

6. Dine Rettigheder

Under GDPR og dansk lovgivning har du følgende rettigheder:

Ret til Indsigt

Du kan anmode om en kopi af alle data vi har om dig.

Ret til Berigtigelse

Du kan anmode om rettelse af forkerte eller ufuldstændige data.

Ret til Sletning

Du kan anmode om sletning af dine personlige data.

Ret til Begrænsning

Du kan anmode om begrænsning af behandlingen af dine data.

Ret til Dataportabilitet

Du kan anmode om dine data i et struktureret, maskinlæsbart format.

Ret til Indsigelse

Du kan gøre indsigelse mod behandlingen af dine data.

For at udøve disse rettigheder, kontakt os på privacy@axionbot.dk.

7. Sikkerhed

Vi tager sikkerhed af dine data meget alvorligt og implementerer følgende foranstaltninger:

Kryptering

Alle data krypteres både under transport og i hvile.

Sikre Servere

Vores servere er placeret i sikre datacentre med fysisk adgangskontrol.

Adgangskontrol

Kun autoriseret personale har adgang til personlige data.

Overvågning

Kontinuerlig overvågning for at opdage og forhindre sikkerhedsbrud.

8. Cookies og Lignende Teknologier

Vi bruger cookies og lignende teknologier til:

8.1 Nødvendige Cookies

  • Autentificering og session management
  • Sikkerhed og fraud prevention
  • Grundlæggende webstedsfunktionalitet

8.2 Analytiske Cookies

  • Forståelse af hvordan brugere interagerer med vores websted
  • Forbedring af brugeroplevelsen
  • Performance optimering

Du kan administrere dine cookie præferencer i din browsers indstillinger.

9. Ændringer til Denne Politik

Vi kan opdatere denne privatlivspolitik fra tid til anden. Væsentlige ændringer vil blive kommunikeret gennem:

  • Email notifikation til registrerede brugere
  • Meddelelse på vores websted
  • Besked i vores Discord server

Vi opfordrer dig til at gennemgå denne politik regelmæssigt for at holde dig opdateret om hvordan vi beskytter dine data.

10. Kontakt Os
